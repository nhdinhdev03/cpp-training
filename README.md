# Khóa Học C++ Chuyên Sâu - 15 Năm Kinh Nghiệm Giảng Dạy

## 👨‍🏫 Về Khóa Học

Đây là khóa học C++ được thiết kế bởi một giảng viên có **15 năm kinh nghiệm** trong việc giảng dạy và phát triển ứng dụng C++. Khóa học không chỉ dạy cú pháp mà còn chia sẻ những kinh nghiệm thực tế, best practices và những cạm bẫy thường gặp trong quá trình lập trình C++.

## 🎯 Mục Tiêu Học Tập

Sau khi hoàn thành khóa học, học viên sẽ:
- Nắm vững các khái niệm cơ bản và nâng cao của C++
- Hiểu sâu về Object-Oriented Programming (OOP) và các design patterns
- Thành thạo Modern C++ (C++11/14/17/20/23)
- Quản lý bộ nhớ hiệu quả và tránh memory leaks
- Sử dụng STL và các thư viện chuẩn một cách chuyên nghiệp
- Viết code thread-safe và hiểu về concurrency
- Áp dụng best practices từ kinh nghiệm thực tế

## 📚 Cấu Trúc Khóa Học

### 📖 Module 1: Nền Tảng C++ (01-fundamentals)
- **Lý thuyết:** Cú pháp cơ bản, biến, hàm, control flow
- **Thực hành:** Các bài tập từ cơ bản đến trung bình
- **Kinh nghiệm:** Debugging tips, coding conventions

### 🏗️ Module 2: Lập Trình Hướng Đối Tượng (02-object-oriented)
- **Lý thuyết:** Classes, Objects, Inheritance, Polymorphism
- **Thực hành:** Thiết kế các hệ thống OOP thực tế
- **Kinh nghiệm:** Design patterns, SOLID principles

### 🚀 Module 3: Khái Niệm Nâng Cao (03-advanced-concepts)
- **Lý thuyết:** Function overloading, operator overloading, friend functions
- **Thực hành:** Xây dựng các class phức tạp
- **Kinh nghiệm:** Khi nào nên và không nên sử dụng các tính năng

### ⚡ Module 4: Modern C++ (04-modern-cpp)
- **Lý thuyết:** C++11/14/17/20/23 features
- **Thực hành:** Smart pointers, move semantics, lambda expressions
- **Kinh nghiệm:** Migration strategies, performance implications

### 🧠 Module 5: Quản Lý Bộ Nhớ (05-memory-management)
- **Lý thuyết:** Stack vs Heap, pointers, references
- **Thực hành:** Memory profiling, leak detection
- **Kinh nghiệm:** Common pitfalls và cách tránh

### 🔧 Module 6: Templates & Generics (06-templates-generics)
- **Lý thuyết:** Function templates, class templates, specialization
- **Thực hành:** Xây dựng generic libraries
- **Kinh nghiệm:** Template debugging, compilation performance

### 📦 Module 7: STL & Containers (07-stl-containers)
- **Lý thuyết:** Vector, map, set, algorithms
- **Thực hành:** Performance comparison, real-world usage
- **Kinh nghiệm:** Choosing right containers, custom comparators

### 🔄 Module 8: Concurrency & Multithreading (08-concurrency)
- **Lý thuyết:** Threads, mutexes, condition variables
- **Thực hành:** Thread-safe programming
- **Kinh nghiệm:** Debugging race conditions, performance tuning

### ✨ Module 9: Best Practices (09-best-practices)
- **Lý thuyết:** Code organization, testing, documentation
- **Thực hành:** Code review exercises
- **Kinh nghiệm:** Industry standards, team collaboration

### 🏆 Module 10: Dự Án Thực Tế (10-projects)
- **Dự án 1:** System Programming Project
- **Dự án 2:** Object-Oriented Design Project
- **Dự án 3:** Modern C++ Application
